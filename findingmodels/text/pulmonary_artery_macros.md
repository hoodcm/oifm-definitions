# Pulmonary artery macros—`OIFM_CDE_000061`

This module describes the common data elements for flow in the pulmonary artery use case

**Codes:** RADELEMENT RDES61 pulmonary artery macros; RADLEX RID874 pulmonary artery

## Attributes

### Flow rate—`OIFMA_CDE_000297`

Measurement of net flow volume in the pulmonary artery above the aortic valve  (mL/sec)  
Mininum: 0  
Maximum: 100  
Unit: ml per sec

### Maximum flow velocity—`OIFMA_CDE_000298`

Measurement of maximum flow velocity in the pulmonary artery above the aortic valve (cm/sec)  
Mininum: 0  
Maximum: 100  
Unit: cm per sec

### Main artery diameter—`OIFMA_CDE_001740`

Diameter of main pulmonary artery in mm, value step 0.1  
Mininum: 0  
Maximum: 100  
Unit: mm

### Left artery diameter—`OIFMA_CDE_001741`

Diameter of left pulmonary artery in mm, value step 0.1  
Mininum: 0  
Maximum: 100  
Unit: mm

### Right artery diameter—`OIFMA_CDE_001742`

Right pulmonary artery diameter in mm, step value 0.1  
Mininum: 0  
Maximum: 100  
Unit: mm

### Diameter ratio—`OIFMA_CDE_001743`

Main pulmonary artery to aortic diameter ratio MPAAo, value step 0.1  
Mininum: 0  
Maximum: 100  
Unit: unit

### Flow ratio—`OIFMA_CDE_001744`

Maximum: 5  
Unit: unit

### Trunk diameter—`OIFMA_CDE_001745`

Measure pulmonary artery trunk dimensions, value step 0.1  
Mininum: 0  
Maximum: 100  
Unit: mm

### Segmental artery diameter—`OIFMA_CDE_001746`

Defines dimensions of lobar and segmental pulmonary arteries, step value 0.1  
Mininum: 0  
Maximum: 100  
Unit: mm

### Measurements evaluation—`OIFMA_CDE_001747`

Compare pulmonary artery measurements to reference values to determine status.

- **enlarged**

- **indeterminate**  
- **unknown**
